# -ZelaBairro---Aplicativo-de-Zeladoria-Urbana
O **ZelaBairro** resolve esse problema oferecendo uma plataforma simples e direta onde qualquer cidadão pode registrar, visualizar, atualizar o status e remover ocorrências de problemas urbanos no seu bairro, promovendo a cidadania ativa e a melhoria do espaço público.
